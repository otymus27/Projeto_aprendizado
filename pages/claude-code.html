🤖 Manual Claude Code

Aprenda a dominar o CLI de IA mais poderoso da atualidade com este guia interativo para iniciantes.

O que é o Claude Code?

Claude Code é uma ferramenta de linha de comando (CLI) desenvolvida pela Anthropic que integra o modelo de linguagem Claude diretamente no seu terminal e ambiente de desenvolvimento. Com ele você pode:

  • Editar e criar arquivos de código com linguagem natural
  • Executar comandos e entender a saída do terminal
  • Navegar e entender codebases inteiras
  • Automatizar tarefas repetitivas de desenvolvimento
  • Revisar, refatorar e depurar código

Instalação

Pré-requisito: Node.js 18+ instalado na sua máquina.

# Instalar globalmente via npm
npm install -g @anthropic-ai/claude-code

# Verificar a versão
claude --version

Primeiros Passos

Após instalar, navegue até seu projeto e inicie o Claude Code:

cd meu-projeto
claude

Na primeira execução, você será solicitado a autenticar com sua conta Anthropic.

Comandos Essenciais

ComandoDescrição
claudeInicia sessão interativa no diretório atual
claude "sua pergunta"Envia uma mensagem direta
claude -p "prompt"Modo não-interativo (pipeline)
/helpLista todos os comandos disponíveis
/clearLimpa o contexto da conversa
/compactComprime o histórico para economizar tokens

Slash Commands

Dentro de uma sessão, utilize os slash commands para ações rápidas:

  • /commit — Cria um commit com mensagem gerada por IA
  • /review — Revisa o código alterado
  • /fix — Tenta corrigir erros identificados
  • /explain — Explica um trecho de código selecionado

Dicas de Uso

Seja específico: Quanto mais contexto você fornecer, melhor será a resposta. Em vez de "corrija esse bug", descreva o comportamento esperado e o atual.
  • Use CLAUDE.md na raiz do projeto para instruções persistentes
  • Prefira editar arquivos existentes a criar novos sempre que possível
  • Utilize /compact em sessões longas para manter performance
  • Explore os hooks para automatizar ações pós-tool
